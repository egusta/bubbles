Vertical Feed Edition

BubbleUp is a modern, TikTok-style web application that delivers endless trivia and educational content in an immersive, vertical video format. It leverages AI to generate content on the fly, fetching images and facts dynamically to keep the user engaged.

âœ¨ Features

Vertical "Infinite" Feed: Swipe up/down to navigate topics, tap to reveal new facts within a topic.

Immersive Visuals: Dynamic, cinematic background images generated by AI (Pollinations.ai/Flux) that match the content.

Magic Creator: A built-in AI tool that allows users to generate custom trivia channels on any topic instantly using OpenAI/Gemini models via proxy.

Mini-Browser: An integrated in-app browser to read source articles (Wikipedia/News) without leaving the experience.

Global Likes (Cloudflare D1): A serverless backend system that tracks "Likes" globally across all users.

Local Persistence: Remembers seen topics, blocked channels, and your personal likes using LocalStorage.

Channel Management: A "Treemap" style dashboard to block/unblock categories and visualize progress.

1. Local / Static Hosting (Quick Start)

You can run BubbleUp directly in any browser without a backend server (Likes will be mocked).

Clone this repository.

Open index.html in your browser.

Done! The app will work immediately. The "Like" count will be simulated locally.

2. Enabling Global Likes (Cloudflare Setup)

To make the "Like" counter real and global for all users, you need to deploy the backend worker.

Prerequisites:

A free Cloudflare account.

Step A: Create the Database

Log in to the Cloudflare Dashboard.

Navigate to Storage & Databases > D1.

Create a new database named trivia-db.

In the Console tab of your new database, run this SQL:

CREATE TABLE IF NOT EXISTS likes (topic_id TEXT PRIMARY KEY, count INTEGER DEFAULT 0);


Step B: Deploy the Worker

Navigate to Compute (Workers).

Create a new Worker (e.g., bubbleup-likes).

Paste the code from worker.js (found in this repo or generated previously) into the online editor.

Go to Settings > Variables and Secrets > D1 Database Bindings.

Add a binding variable named DB and connect it to your trivia-db.

Deploy the worker.

Step C: Connect Frontend

Copy your Worker's URL (e.g., https://bubbleup-likes.yourname.workers.dev).

Open index.html.

Find the line const API_URL = "";.

Paste your URL: const API_URL = "https://bubbleup-likes.yourname.workers.dev";.

Save and host index.html anywhere (GitHub Pages, Vercel, Netlify, etc.).
